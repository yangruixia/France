# Si的四个用法

si = if
引导条件从句
表示：如果……就……

si做副词，起强调作用，表示为：如此地、这般地
Il était si patience, si aimable.
他是那么耐心，那么和蔼可亲。

si引导的从句在这里不表示假设，而是起转折和强调主语的作用。
Si Paris est le coeur de la france, l'ile de cité est le coeur de Paris.
巴黎是法国的中心，而西岱岛则是巴黎的中心。
